```md
Tauri is an open-source framework designed for building cross-platform desktop applications using web technologies. It enables developers to create applications using languages like JavaScript, TypeScript, or Rust while leveraging web front-end technologies such as HTML, CSS, and any JavaScript framework like React, Vue, or Svelte. One of Tauri's key features is its ability to produce highly optimized, lightweight binaries that consume minimal resources compared to other frameworks like Electron.

Tauri applications use a Rust-based backend for performance and security, while the frontend is typically based on web technologies, allowing for seamless integration with existing web development workflows. This makes Tauri an appealing choice for developers seeking to create desktop applications without sacrificing performance or resource usage.

The Tauri framework focuses on enhancing user privacy and security by minimizing data collection and providing developers with fine-grained control over application permissions. Additionally, Tauri supports automatic updates, file system access, and other native functionality across platforms such as Windows, macOS, and Linux.
```

```md
Tauri 是一个开源框架，旨在使用网页技术构建跨平台的桌面应用程序。它使开发者能够使用 JavaScript、TypeScript 或 Rust 等语言，同时利用诸如 HTML、CSS 以及 React、Vue 或 Svelte 等任何 JavaScript 框架来创建应用程序。Tauri 的一个关键特点是能够生成高度优化的轻量级二进制文件，与 Electron 等其他框架相比，资源占用极少。

Tauri 应用程序采用基于 Rust 的后端以提升性能和安全性，而前端通常基于网页技术，实现与现有网页开发工作流的无缝集成。这使得 Tauri 成为希望创建桌面应用且不牺牲性能或资源使用的开发者的理想选择。

Tauri 框架注重提升用户隐私和安全，尽量减少数据收集，并为开发者提供对应用权限的细粒度控制。此外，Tauri 支持自动更新、文件系统访问以及在 Windows、macOS 和 Linux 等多个平台上的其他本地功能。
```
