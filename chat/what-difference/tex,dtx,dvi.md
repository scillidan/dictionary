What is the difference between tex, dtx, dvi file?

```md
The file formats TeX, DTX, and DVI serve different purposes within the TeX typesetting system. Here’s a brief overview of each:

1. **TeX (.tex)**:
   - **Description**: This is a plain text file that contains the source code written in the TeX typesetting language. It consists of commands and text that instruct the TeX engine how to format and typeset the content.
   - **Usage**: Users write their documents in this format using a text editor. The TeX file includes formatting commands that control layout, fonts, sections, and other document features.

2. **DTX (.dtx)**:
   - **Description**: DTX files are specialized source files that contain both documentation and code of a TeX package or class. It allows authors to bundle the package's source code with its documentation in a single file.
   - **Usage**: When processed, DTX files typically generate both a .sty or .cls file (which contains the package or class code) and a `.pdf` or `.dvi` file containing the documentation. This is useful for distributing packages along with their explanations and examples.

3. **DVI (.dvi)**:
   - **Description**: DVI stands for DeVice Independent. It is the output format generated by the TeX typesetting engine after processing a .tex file. This format does not contain any font or image data but instead references them, allowing it to be independent of the output device.
   - **Usage**: DVI files can be converted to other formats (like PDF or PostScript) for printing or viewing. They are typically used as an intermediate file format in the typesetting workflow.

### Summary:
- **.tex** is the source document you write.
- **.dtx** is a dual-purpose file containing both documentation and source code for packages.
- **.dvi** is the intermediate output format produced by the TeX engine, ready for further processing to produce final output formats like PDF.
```

```md
以下是对内容的中文翻译：

TeX、DTX 和 DVI 文件格式在 TeX 排版系统中发挥着不同的作用。以下是对每种格式的简要概述：

1. **TeX (.tex)**:
   - **描述**：这是一个纯文本文件，包含用 TeX 排版语言编写的源代码。它由指令和文本组成，指示 TeX 引擎如何格式化和排版内容。
   - **用法**：用户使用文本编辑器以此格式撰写文档。TeX 文件包括控制布局、字体、章节和其他文档特性的格式化命令。

2. **DTX (.dtx)**:
   - **描述**：DTX 文件是特殊的源文件，同时包含 TeX 包或类的文档和代码。它允许作者将包的源代码与其文档打包到一个文件中。
   - **用法**：处理时，DTX 文件通常会生成一个 .sty 或 .cls 文件（包含包或类代码）和一个包含文档的 `.pdf` 或 `.dvi` 文件。这对于分发包及其说明和示例非常有用。

3. **DVI (.dvi)**:
   - **描述**：DVI 代表设备无关（DeVice Independent）。这是 TeX 排版引擎在处理 .tex 文件后生成的输出格式。该格式不包含任何字体或图像数据，而是引用它们，允许其独立于输出设备。
   - **用法**：DVI 文件可以转换为其他格式（如 PDF 或 PostScript）以进行打印或查看。它们通常用作排版工作流程中的中间文件格式。

### 总结：
- **.tex** 是你编写的源文档。
- **.dtx** 是一个双重用途文件，包含包的文档和源代码。
- **.dvi** 是 TeX 引擎生成的中间输出格式，准备进一步处理以生成最终输出格式，如 PDF。
```
