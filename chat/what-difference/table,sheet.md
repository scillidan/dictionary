What is the difference between table, sheet?

```md
The terms "table" and "sheet" are often used in contexts such as databases, spreadsheets, and data organization, but they refer to different concepts. Here’s a breakdown of the differences:

### Table
1. **Definition**: A table is a structured arrangement of data in rows and columns. Each row represents a single record or entry, while each column represents a specific attribute or field of that record.
  
2. **Usage**: Tables are commonly used in databases (like SQL databases) and document formats (like HTML) to organize and present data clearly. They can also be found in reports and forms.

3. **Example**: In a database of employees, a table might have columns for employee ID, name, department, and salary, with each row corresponding to a different employee.

### Sheet
1. **Definition**: A sheet (often referred to as a "worksheet") usually indicates a single page or section within a spreadsheet application (such as Microsoft Excel or Google Sheets). A sheet can contain multiple tables, along with other data types, charts, and graphs.

2. **Usage**: Sheets are part of a larger workbook (or spreadsheet file) and can contain numerous data points and visual representations. Users can work with multiple sheets within the same workbook to organize related but distinct sets of data.

3. **Example**: In a budget spreadsheet, one sheet might be dedicated to monthly expenses, while another sheet may focus on income, and a third may contain summary calculations.

### Summary
- **Table**: A specific data structure (rows and columns) used within databases and documents for data organization.
- **Sheet**: A page within a spreadsheet application that can house multiple tables and various data types.

In short, a table is a format for organizing data, whereas a sheet is a broader concept that refers to a space where data can be displayed and manipulated, often containing one or more tables.
```

```md
术语“表格”和“工作表”通常在数据库、电子表格和数据组织等上下文中使用，但它们指的是不同的概念。以下是两者的区别：

### 表格
1. **定义**：表格是以行和列的结构方式排列的数据。每一行代表一个单独的记录或条目，而每一列则代表该记录的特定属性或字段。

2. **用途**：表格通常用于数据库（如 SQL 数据库）和文档格式（如 HTML）中，以清晰地组织和展示数据。它们也常出现在报告和表单中。

3. **示例**：在一个员工数据库中，表格可能有员工 ID、姓名、部门和工资的列，每一行对应一位不同的员工。

### 工作表
1. **定义**：工作表（通常称为“工作单”）通常指的是电子表格应用程序（如 Microsoft Excel 或 Google Sheets）中的单个页面或部分。一个工作表可以包含多个表格，以及其他数据类型、图表和图形。

2. **用途**：工作表是更大工作簿（或电子表格文件）的一部分，可以包含大量数据点和可视化表示。用户可以在同一工作簿中使用多个工作表来组织相关但不同的数据集。

3. **示例**：在预算电子表格中，一个工作表可能专注于每月支出，而另一个工作表可能关注收入，第三个工作表则可能包含汇总计算。

### 总结
- **表格**：一种特定的数据结构（行和列），用于数据库和文档中的数据组织。
- **工作表**：电子表格应用程序中的一个页面，可以容纳多个表格和各种数据类型。

简而言之，表格是一种组织数据的格式，而工作表是一个更广泛的概念，指的是一个可以展示和操作数据的空间，通常包含一个或多个表格。
```
